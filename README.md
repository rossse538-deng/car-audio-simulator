# 🚗 车载低音单元仿真分析引擎 V4.2 (Car Audio Subwoofer Simulator)

基于客观电声学定律与非对称声学对齐理论开发的车载音响调音辅助工具。本引擎纯前端运行，旨在帮助汽车音响改装者、调音师及爱好者通过科学的推演，评估车门中低音单元的物理潜能，并制定合理的 DSP（数字音频处理器）分频策略。

## ✨ 核心功能与亮点

- **📐 核心物理与数学计算模型**
  - **霍夫曼铁律 (Hofmann's Iron Law) 校验：** 自动排查扬声器参数（灵敏度、谐振频率、等效容积）是否存在物理悖论，还原真实的喇叭素质。
  - **动态装车阻尼估算 (Lossy Baffle)：** 将车门视为带泄露因子的有限大障板，引入气密系数推演喇叭装车后的真实阻尼 (Qtc) 和共振峰值 (Fc)。
  - **未知参数推演：** 即使没有官方 T/S 参数，仅凭万用表实测 Fs 或标称数据，也能推断出扬声器的设计取向（如 Free-Air、密闭箱取向等）。
  
- **🎛️ 非对称声学 DSP 对齐策略**
  - 根据喇叭的物理极限与车门环境，自动计算并推荐 **高通 (HPF)** 与 **低通 (LPF)** 的频率及斜率。
  - 融合自然声学滚降与电学滤波，实现近似 Linkwitz-Riley (LR4) 的相位同向衔接。

- **📋 系统综合诊断报告**
  - 一键生成并导出包含系统 Qtc、架构意图、DSP 建议、车门隔音建议的综合诊断报告图片（自带防伪水印）。

## 🚀 如何使用

本项目为纯静态 HTML 页面，无需安装任何环境或依赖：

1. **在线使用 (推荐)：** 直接访问[👉 点击这里进入在线仿真引擎](https://rossse538-deng.github.io/car-audio-simulator/) 
2. **本地使用：** 下载本仓库的 `index.html` 文件，双击使用任意现代浏览器（Chrome, Edge, Safari）打开即可。

## 🛠️ 操作流程

1. **设定声学边界条件：** 选择您的车型门板容积、车门隔音处理状态以及是否有独立超低音。
2. **输入扬声器参数：** 
   - 如果您有官方说明书，请选择“已知准确 T/S 参数”并输入 Fs、Qts、Vas。
   - 如果没有参数，可使用万用表测出峰值电压频率，或输入标称参数，引擎将自动结合物理定律进行推演。
3. **执行计算：** 点击“执行声学仿真与 DSP 计算”，下方将生成详细的物理数据、设计取向评估及 DSP 分频建议。
4. **导出报告：** 点击“📸 导出分析报告”按钮，将结果保存为长图以便分享或存档。

## 💻 技术栈

- 纯原生 HTML5 / CSS3 / Vanilla JavaScript
- 响应式 UI 设计，兼容手机与电脑端浏览
- 报告截图导出采用 [html2canvas](https://html2canvas.hertzen.com/) 库支持

## ⚠️ 免责声明

本引擎基于理想化的物理与数学声学模型推演（如 Lossy Baffle 与 Acoustic LR4）。由于真实车厢存在驻波、反射及相干干涉等极其复杂的声学环境，本引擎输出的 DSP 频点与 Qtc 评估仅供**工程参考与调音基准起步**，最终效果请以 RTA 实时声学测试与实际听感为准。
