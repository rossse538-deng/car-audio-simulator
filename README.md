# 🚗 车载低音单元与门板适配分析系统 (Car Audio Acoustic Coupling Engine)

> 一个基于“有损无限大障板 (Lossy Infinite Baffle)”物理模型的汽车音响工程级诊断与仿真引擎。

## 💡 设计初衷：打破汽车音响的“调音万能论”

在汽车音响改装行业存在一个普遍的误区：**玩家往往先根据品牌、价格或主观听感选择中低音单元，装车后再试图通过 DSP (数字信号处理) 努力“调试”出最佳状态。**

然而，声学是一门严谨的物理科学。后期的电子调试（EQ、延时、相位）永远无法打破扬声器的**物理工作边界**。
* 如果喇叭的设计参数（T/S 参数）强依赖密封箱体，将其装在漏气的车门里，大动态时必然“拍边”失控。
* 如果喇叭的谐振点偏高，在配合低音炮时，必然会导致前场低频基频缺失，低频结像彻底跑回后备箱。

**正确选择低音的逻辑，不应该是根据品牌，而应根据“机电参数与车门物理边界的耦合度”。** 否则选择错了喇叭，再顶级的 DSP 和调音师也无法解决根源问题。

本项目正是基于此痛点开发。它能让玩家在购买/安装喇叭之前，通过输入 T/S 参数，在工程级物理模型下仿真其装车后的真实表现，避免花钱踩坑。

---

## 🔬 核心物理内核 (The Truth)

本引擎**拒绝使用**传统的“密闭箱 $Q_{tc}$ 倒推公式”，因为车门内部充满孔洞与排水口，绝非密闭箱。

1. **真实边界模型 (Lossy IB)**：引擎采用 `Q_sys = Q_ts × √(1 + α × L_f)` 修正泄漏系数模型，将车门定义为“带有声短路效应的有损无限大障板”，计算出的系统阻尼与下潜极度贴近实车 DATS 仪器实测值。
2. **实战双分支评估架构**：
   * **【有超低音系统】**：引入空间心理声学定位实战经验。苛刻评估门板中低音的自然滚降点 ($F_{sys}$) 是否能与 DSP 的 50-60Hz 高通滤波完美叠加，从而产生平滑的林奎茨-瑞利 (LR4) 声学交越，将后备箱低频能量强力“拉”至前场仪表台。
   * **【无超低音系统】**：严苛审查门板喇叭在全频直推状态下的极低频下潜能力 ($F_{sys}$) 与动态阻尼 ($Q_{sys}$) 厚度自补偿能力。
3. **灵敏度驱动预警**：独立评估喇叭的微信号发声能力，防止低灵敏度重型喇叭被原车主机推至失真。

---

## 🚀 核心功能 (Features)

* 🎛️ **极简参数输入**：仅需输入喇叭说明书上的 5 个核心参数（$F_s, Q_{es}, Q_{ms}, V_{as}$, 灵敏度）。
* 🚘 **车门声学匹配**：内置车型容积估算与隔音耗散系数模型。
* 📊 **动态阻尼预推演**：自动计算装车后的系统品质因数 ($Q_{sys}$) 和谐振点 ($F_{sys}$)。
* 🩺 **去营销化诊断报告**：摒弃“好听/难听”的玄学词汇，输出诸如“驻波过载”、“声学交越盲区”、“结像后移”等纯粹的客观工程建议。
* 📸 **一键生成仿真报告**：支持无损渲染截图，方便分享给调音师或车友探讨。
